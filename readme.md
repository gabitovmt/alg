# Структуры данных и алгоритмы

## Структуры данных и алгоритмы Java, 2-е издание, Роберт Лафоре

- `/src/alg` - код из книги.
- `ch*` - главы.
- `ex*` - примеры из глав.
- `home*` - домашняя работа (упражнения, программные проекты).
- `ext*` - мой код.
- `/src/workshop` - портировал приложения *Workshop из апплетов в настольные Swing приложения

### Найденные ошибки

Стр. 95. Метод `bubbleSort()`. Необходимо в условие внешнего цикла написать `out>0`, а не `out>1`.
Для проверки, попробуй отсортировать массив `[3, 2, 1]`. С условием `out>1` он не будет отсортирован.
